# Acesso ao Conjunto de Dados (Data Access)

O conjunto de dados original utilizado para gerar os resultados e as conclusões apresentadas neste estudo não se encontra nesta pasta.

Devido a acordos de confidencialidade com o projeto Debaqi e para garantir a privacidade dos participantes, estes dados não podem ser disponibilizados publicamente.

## Dados de Exemplo

Para permitir a execução do código e a demonstração da nossa metodologia, foi criado um conjunto de dados de exemplo (`sample_data`). Pode encontrar este ficheiro na pasta `/sample_data`, localizada na raiz do repositório.

## Utilizar os Seus Próprios Dados

Se desejar utilizar o seu próprio conjunto de dados, certifique-se de que este segue a mesma estrutura do nosso, conforme detalhado no manual de anotação (`annotation_manual.pdf`), disponível na pasta `/annotation_materials`.
